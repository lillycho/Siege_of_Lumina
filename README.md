# Lumina Tetra Defense Demo

`system_design.md`의 단일 스테이지 규칙과 Figma 프레임 3 HUD 배치를 바탕으로 만든 웹 플레이 데모입니다. 외부 패키지나 빌드 과정 없이 Node.js만으로 실행됩니다.

## 실행

```powershell
npm run dev
```

브라우저에서 `http://127.0.0.1:4173`을 엽니다.

## 테스트

```powershell
npm test
```

## 조작

- 일반 카드 선택: 카드 클릭
- 배치: 강조된 전선의 파란 열 번호 클릭
- 회전: `R` 키 또는 화면의 R 버튼
- 패스: `이번 턴 패스` 버튼
- 이벤트 카드: 좌하단 카드 클릭 후 필요한 대상 선택

## 현재 구현 범위

- 좌우 2방향 6×10 전선과 시작칸 대기열
- 20턴, 성 HP, 승리·패배 판정
- I/T/S/Z/J/L 블럭과 나무·돌·철 확률
- 방향 제한, 회전 자원, 자동 낙하와 충돌 판정
- 적 이동·블럭 공격·성 피해·3턴 스폰 예고
- 나무→돌→철 2×2 합성과 합성폭발 보상
- 이벤트 카드 20종
- 1440×1024 기준 중앙 좌우 전장 HUD
- 고정 시드 재현: `?seed=1234`

임시 도형과 CSS 그래픽을 사용했으며, 최종 도트 아트와 사운드는 이후 교체할 수 있도록 게임 규칙과 화면 표현을 분리했습니다.
